# Локальная админка СМУ-1 (Stage 1)

## Как открыть visual editor
1. Запустите фронтенд: `npm run dev -- --host 127.0.0.1`.
2. Запустите admin API: `npm run admin:api`.
3. Откройте `http://127.0.0.1:4321/SMU1/admin/`.
4. Войдите (`admin / admin`).
5. Нажмите **«Редактировать страницу Навесы»**.
6. Откроется основной visual edit mode: `http://127.0.0.1:4321/SMU1/admin/pages/navesy/`.

## Основной visual editor
Режим сделан как «обычная страница + инструменты редактирования»:
- фиксированная верхняя панель (`← В админку`, статус, `Сохранить`, `+ Добавить секцию`, `Открыть публичную страницу`, `Технический режим`, `Выйти`);
- контент отображается как публичная страница;
- редактор работает inline в блоках;
- внизу оставлен только debug-блок `Показать JSON`.

## Технический режим (запасной)
Технический редактор не является основным сценарием.
Используется только как fallback по ссылке **«Технический режим»** из visual editor.

## Как редактировать hero
1. Нажмите `Редактировать hero`.
2. Изменяйте заголовок, описание и подпись placeholder прямо в hero.
3. Доступны действия:
   - `Заменить фото`;
   - `Убрать фото`;
   - `Настроить кадр`;
   - `Готово` / `Отмена`.

## Как редактировать карточку
В блоках `cardGrid`/`exampleGrid` у карточек доступны:
- `Редактировать`;
- `Выше` / `Ниже`;
- `Скрыть`;
- `Удалить`.

При `Редактировать` форма открывается внутри карточки: `Заголовок`, `Описание`, `Активна`, `Готово`, `Отмена`.

## Как добавить карточку
Внизу блока карточек нажмите `+ Добавить карточку`.
Новая карточка добавляется в конец и сразу открывается в inline-редактировании.

## Как удалить карточку
Нажмите `Удалить` на карточке.
Появляется подтверждение:
`Удалить карточку? Это действие применится после сохранения страницы.`

До нажатия `Сохранить` удаление только локальное в draft.

## Как добавить секцию
Кнопка `+ Добавить секцию` в верхней панели открывает модальное окно.
Выберите тип:
- Карточки;
- Список;
- Процесс;
- Факторы стоимости;
- Текст + изображение;
- Вставка;
- Примеры.

Секция создается с дефолтными полями stage-1 и добавляется в конец `pageBlocks`.

## Как удалить секцию
У секции нажмите `Удалить секцию`.
Подтверждение:
`Удалить секцию? Это действие применится после сохранения страницы.`

## Как открыть настройки секции
Нажмите `Настройки` в панели секции.
Откроется fixed modal поверх экрана.
Поля:
- Заголовок;
- Intro;
- Тема;
- Сетка (если поддерживается);
- Колонки (если поддерживаются);
- Активность.

Закрытие: `Готово`, `Отмена`, `Esc`, клик вне окна.

## Как редактировать списки и process
- `listPanel`/`factorList`: `Редактировать список` → inline-строки с `Выше/Ниже/Удалить` + `+ Добавить строку`.
- `process`: шаги редактируются прямо в блоке, можно `Добавить`, `Удалить`, `Скрыть`, менять порядок.

## Как загрузить фото
Используется endpoint:
- `POST /api/admin/upload`

Ограничения:
- только после авторизации;
- `multipart/form-data`;
- типы: `jpg`, `jpeg`, `png`, `webp`, `svg`;
- до `10 MB`.

Файлы сохраняются в:
- `public/uploads`

Ответ:
- `{ "path": "/uploads/file-name.ext" }`

## Как убрать фото
Кнопка `Убрать фото` очищает `image`/`media` и возвращает placeholder.

## Как настроить кадрирование
Кнопка `Настроить кадр` открывает модальное окно с настройками:
- Fit (`Заполнить`/`Вписать`);
- Масштаб;
- Сдвиг X;
- Сдвиг Y.

Настройки пишутся в:
- `imageView` (для hero);
- `mediaView` (для блоков).

## Сохранение
Кнопка `Сохранить` отправляет:
- `PUT /api/admin/content/product-sections/navesy`

После успеха:
- статус: `Нет изменений`;
- показывается `Сохранено`;
- без полной перезагрузки страницы.
